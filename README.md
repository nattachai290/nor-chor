# 🎮 Nor-Chor

Tools & Calculators for games.

🔗 **https://nattachai290.github.io/nor-chor/**

## Games

### 💎 ENJ Excavators
Calculator สำหรับเกม ENJ Excavators — เปรียบเทียบความคุ้มค่าของการ +ตัว vs +% ทุก Goblin พร้อม Countdown Season, Save/Load data และ Google Drive sync

### 🃏 P5X Build Optimizer
Build Reference และ Stat Calculator สำหรับ Persona 5: The Phantom X (Global) — เลือกตัวละคร ดู card sets, weapon, stat priority และคำนวณ build score พร้อม filter ตาม Role และ Element
